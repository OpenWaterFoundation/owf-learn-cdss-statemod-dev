Revision Control

Revision control using Git/Github is used to track changes to the software. Revision control provides many benefits, in particular when working with multiple software developers.

StateMod code is maintained in GitHub (add URL when finalized).

Any developer that actively edits StateMod code must adhere to revision control protocols.
